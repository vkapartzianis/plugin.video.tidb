### To release a new version of the addon, follow these steps:

1. Update the version number in `plugin.video.tidb.kine/addon.xml` (also update the `<path>` and `<news>` entries).
2. Zip the `plugin.video.tidb.kine` folder as `plugin.video.tidb.kine-[version].zip`. The zip must contain the `plugin.video.tidb.kine/` folder at its top level (Kodi's "Install from zip file" expects this).
3. Bundle the updated add-on with Kine, or install the zip manually in stock Kodi via **Settings → Add-ons → Install from zip file**.

The add-on is distributed by bundling it with Kine; there is no standalone Kodi repository.
